# 🍅 Pomodoro Timer

Простой таймер-помодоро на **React + Vite**.

## Возможности

- Три режима: **Помидор** (25 мин), **Короткий перерыв** (5 мин), **Длинный перерыв** (15 мин)
- Старт / Пауза / Сброс (старт/паузу можно ставить пробелом)
- Круговой индикатор прогресса (SVG)
- Звуковой сигнал по завершении (Web Audio API)
- Счётчик завершённых «помидоров»
- Цвет темы меняется под текущий режим

## Запуск

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```bash
npm install     # установить зависимости
npm run dev     # запустить дев-сервер
```

Открой адрес из консоли (обычно http://localhost:5173).

## Сборка

```bash
npm run build     # собрать в папку dist/
npm run preview   # локально посмотреть собранную версию
```

## Структура проекта

```
pomodoro/
├── index.html                 # точка входа Vite
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx               # монтирование React
    ├── App.jsx                # корневой компонент
    ├── index.css              # стили
    ├── constants.js           # настройки режимов
    ├── components/
    │   ├── ModeTabs.jsx       # переключатель режимов
    │   ├── TimerDisplay.jsx   # круговой таймер
    │   └── Controls.jsx       # кнопки Старт/Пауза/Сброс
    ├── hooks/
    │   └── usePomodoro.js     # логика таймера
    └── utils/
        ├── formatTime.js      # форматирование MM:SS
        └── playBeep.js        # звуковой сигнал
```
